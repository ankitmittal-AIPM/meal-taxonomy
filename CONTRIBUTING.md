## 2. Code Guidelines
- Use Python 3.10+  
- Follow modular folder structure  
- Keep functions pure when possible  
- Add logs (structured format)  
- Add docstrings  

## 3. Adding Tag Types
Use `tag_types` table:  
- id  
- label  
- description  

## 4. Pull Request Checklist
- Clear description  
- Working tests  
- No breaking code  
- Use meaningful commit messages
